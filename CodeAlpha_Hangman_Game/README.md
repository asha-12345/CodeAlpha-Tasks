
Hangman Game 🎮

Description

This is a simple text-based Hangman Game developed using Python. The player must guess a hidden word one letter at a time. The game randomly selects a word from a predefined list of five words. Players have a maximum of 6 incorrect attempts to guess the word correctly before the game ends.

Features

- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Displays guessed letters
- Win/Lose result display
- Simple console-based interface

Technologies Used

- Python
- Random Module
- Loops (while)
- Conditional Statements (if-else)
- Strings
- Lists

How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Run the Python file:
   python hangman_game.py

Learning Outcomes

This project helps beginners understand:

- Python basics
- Loops and conditions
- String manipulation
- List operations
- User input handling
