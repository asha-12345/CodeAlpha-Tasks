
Stock Portfolio Tracker 📈
Description

This is a simple Stock Portfolio Tracker developed using Python. The program allows users to add stocks, enter the number of shares owned, and calculate the total value of their investment portfolio. It helps users keep track of their stock holdings in an easy and efficient way.

Features

- Add multiple stock holdings
- Enter stock names and quantities
- Calculate total portfolio value
- Display portfolio summary
- Simple console-based interface
- Easy-to-understand implementation

Technologies Used

- Python
- Dictionaries
- Loops (for, while)
- Conditional Statements (if-else)
- User Input Handling
- Basic Arithmetic Operations

How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Run the Python file:
   python stock_portfolio_tracker.py

Learning Outcomes

This project helps beginners understand:

- Python basics
- Dictionary operations
- Loops and conditions
- Data storage and retrieval
- User input handling
- Portfolio value calculation
